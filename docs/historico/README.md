# Historico

Esta pasta preserva documentos, dashboards e scripts anteriores do projeto. Os arquivos foram movidos para fora da raiz para manter o repositorio publicado mais legivel no GitHub.

## Subpastas

- `artefatos_gerados/`: HTMLs antigos gerados durante rodadas anteriores.
- `dashboard_obsoleto/`: copia do dashboard antigo movido para preservacao.
- `scripts_anteriores/`: scripts substituidos pelo pipeline atual.

## Arquivos principais

- `00_LEIA_PRIMEIRO*.md`: notas de orientacao de rodadas anteriores.
- `*_DASHBOARD*.md`: auditorias, checklists e registros de evolucao do painel.
- `DIAGNOSTICO_*.md`: diagnosticos metodologicos e de dados.
- `METODO*.md`, `MODELO*.md` e `TIPOLOGIA_*.md`: documentos conceituais e tecnicos preservados.
- `RELATORIO_*.md`, `REVISAO_*.md` e `ROTEIRO_*.md`: registros de validacao, revisao e apresentacao.
- `01_INVENTARIO_ARQUIVOS.csv`: inventario historico dos arquivos antes da reorganizacao.

Os scripts ativos ficam em `scripts/` na raiz. Os agregados publicos atuais ficam em `assets/` e `assets/referencia/`.
