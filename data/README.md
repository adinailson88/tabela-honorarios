# data

Diretório reservado para organização futura dos dados do projeto.

## Subpastas

- `public/`: dados agregados que podem ser publicados no GitHub Pages.
- `local/`: dados locais, brutos ou linha a linha, que não devem ser publicados.

## Regra de segurança

Dados individualizados de ARTs não devem ser publicados neste repositório, salvo decisão formal documentada e revisão metodológica específica.

Em caso de dúvida, manter o arquivo fora da publicação e registrar:

`Informação insuficiente para verificar.`
